Artikler

MathML

MathML står for Mathematical Markup Language, og er et verktøy brukt for å representere matematiske formler på nettsider og andre dokumenter. Det finnes to forskjellige typer, nemlig C-MathML og P-MathML. Det er blitt en del av HTML5, men ikke alle nettlesere støtter det likevel.

SVG

SVG står for Scaleable Vector Graphics og er XML-basert. Det er et verktøy brukt for 2D vektorbasert bildeformat, og er også blitt en standard del av HTML5. SVG kan skaleres uendelig uten å påvirke kvalitet, iforhold til vanlige bildeformat som .jpg og .png.

GitHub

GitHub er et distribuert oppbevaringssted / versjonskontrollsystem som blir brukt av webutviklere for enklere samarbeid. På dette systemet kan man dele kode/filer med hverandre, gjennopprette tidligere versjoner, få full oversikt over historie til filer, samt backup av filer.
