Git og GitHub

Hva er Git og GitHub?

Git GitHub er et distribuert oppbevaringssted / versjonskontrollsystem som blir brukt av webutviklere for enklere samarbeid. På dette systemet kan man dele kode/filer med hverandre, gjennopprette tidligere versjoner, få full oversikt over historie til filer, samt backup av filer.
